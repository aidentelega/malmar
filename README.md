## 🎯 Purpose of this software

Malmar helps you organize files on your computer. Many users struggle with cluttered folders and lost documents. This tool scans your storage drives, identifies duplicates, and moves files into logical folders based on their content. You save time and find your documents without searching through dozens of subdirectories. The software runs locally on your machine to ensure your data stays private.

## 💻 System requirements

Before you begin, ensure your computer meets these basic needs:

* Windows 10 or Windows 11.
* At least 200 megabytes of free disk space.
* An active internet connection for the initial download.
* Administrator access for your user account.

## 🚀 Running your first scan

You can start the program from your desktop. Double-click the Malmar icon to open the main window. 

1. Select the drives or folders you want to organize. Click the Add Folder button to pick a specific location on your computer.
2. Choose your preferred sorting method. You can sort by date, by file type, or by name.
3. Click the Scan button. The software analyzes your files. This takes time if you have many files. 
4. Review the results on the screen. Malmar suggests a new structure for your files.
5. Click Run to apply the changes. The software moves the files into their new folders.

We built this software with your safety in mind. Malmar does not upload your personal files to any server. All processing happens on your local hardware. No one else gains access to your documents during the organization process. If you decide to remove the software, you can uninstall it from your Windows settings at any time without leaving behind active background tasks.

## 🔍 Understanding the interface

The main window shows three distinct areas. 

* The Left Sidebar displays your selected folders. You can add or remove locations here.
* The Center View shows the proposed file changes. The software groups files that belong together. You can uncheck any file if you prefer to keep it in the original location.
* The Right Panel provides status updates. It shows progress bars when the tool scans or moves files. It also displays a log if the tool encounters a locked file or a permission error.

## 📋 Troubleshooting common issues

If you have trouble, check these common fixes:

* Software does not open: Ensure you installed the latest version from the releases page. Restart your computer if a background process hangs.
* Access denied: Some folders are system-locked. Run the software as an administrator by right-clicking the icon and choosing Run as Administrator.
* Files do not move: Check if you have enough space on the target drive. Also, verify that the files are not currently open in another program like Microsoft Word or Excel.
* Scan takes too long: If you have thousands of photos or high-definition videos, the initial scan takes longer. Leave the computer running and let the software finish the task.

## 📑 Managing file conflicts

Sometimes two files have the same name in different folders. Malmar handles these conflicts automatically. If a conflict occurs, the software appends a number to the file name so you do not lose any data. You can find these files in a generated folder called `Conflicts` located inside your target directory. Always check this folder after a large move operation.
